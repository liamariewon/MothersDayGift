# Grocery App

Local test project for turning recipe URLs into a combined grocery list.

## Files
- `app.py` - Streamlit interface
- `scraper.py` - recipe fetching and ingredient parsing logic
- `requirements.txt` - Python packages to install
- `sample_urls.txt` - starter URL list for testing

## Setup
```bash
python -m pip install -r requirements.txt
```

## Run
```bash
python -m streamlit run app.py
```

## Notes
- Do not run with the VS Code play button.
- Run from a terminal.
- If the app file is not in your current folder, use the full path.
